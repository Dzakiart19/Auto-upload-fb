Tujuan: Perbaiki error unggah video Facebook "File video tidak valid atau corrupt" dengan menambahkan langkah konversi video menggunakan FFmpeg.

Lokasi Perbaikan:
1.  **Workflow Utama:** `src/mastra/workflows/facebookVideoWorkflow.ts`
2.  **Tool Baru:** Buat tool baru bernama `ffmpegConvertVideo.ts`

Detail Perbaikan:

**Langkah 1: Buat Tool Baru `ffmpegConvertVideo.ts`**
Buat tool baru di `src/mastra/tools/ffmpegConvertVideo.ts` dengan fungsi:
-   Input: `videoPath` (string, path file lokal video asli).
-   Output: `convertedVideoPath` (string, path file lokal video hasil konversi).
-   Logika: Jalankan perintah `ffmpeg` untuk mengonversi video dari `videoPath` ke `convertedVideoPath`.
    -   Gunakan perintah: `ffmpeg -i [input_path] -c:v libx264 -preset fast -c:a aac -strict experimental [output_path]`
    -   Pastikan output file disimpan di direktori `/tmp/telegram_videos` (sama dengan `telegramDownloadVideo.ts`).
    -   Jika konversi gagal, kembalikan `success: false` dengan pesan error dari `ffmpeg`.

**Langkah 2: Modifikasi Workflow `facebookVideoWorkflow.ts`**
Di dalam fungsi `processVideoDirectly` (atau di dalam `facebookVideoAgent.generateLegacy` jika AI aktif), ubah urutan eksekusi *tool* menjadi:

1.  **Download Video:** Panggil `telegramDownloadVideo` untuk mendapatkan `originalVideoPath`.
2.  **Konversi Video (BARU):** Panggil `ffmpegConvertVideo` dengan `originalVideoPath` sebagai input. Dapatkan `convertedVideoPath`.
3.  **Upload Video:** Panggil `facebookUploadVideoSmart` (atau `facebookUploadVideo`) menggunakan **`convertedVideoPath`** sebagai input `videoPath`.
4.  **Pembersihan:** Setelah unggahan selesai (berhasil atau gagal), pastikan kedua file (`originalVideoPath` dan `convertedVideoPath`) dihapus dari sistem file.

**Langkah 3: Modifikasi Tool `facebookUploadVideoSmart.ts`**
Tambahkan logika pembersihan file di akhir eksekusi `facebookUploadVideoSmart.ts` (atau `facebookUploadVideo.ts` jika `Smart` tidak digunakan) untuk menghapus file video yang diunggah. Ini penting karena file yang diunggah sekarang adalah file hasil konversi.

Dengan perubahan ini, setiap video dari Telegram akan dipaksa ke format yang kompatibel sebelum diunggah ke Facebook, yang akan menyelesaikan error "File video tidak valid atau corrupt".
